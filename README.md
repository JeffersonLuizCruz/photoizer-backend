# CarolCRM

Sistema de gestão de ensaios fotográficos.

## Swagger UI

Acesse a documentação interativa da API (Swagger UI) em:

```
http://localhost:8080/swagger-ui.html
```

Ou via OpenAPI JSON:

```
http://localhost:8080/v3/api-docs
```

## Console H2

O banco de dados H2 em memória pode ser acessado pelo console web em:

```
http://localhost:8080/h2-console
```

Configuração para conexão:

- **JDBC URL**: `jdbc:h2:mem:crmdb`
- **User Name**: `sa`
- **Password**: *(deixe em branco)*
